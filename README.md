# BSCTest


Платежная система
=================

Основная информация
-------------------

На вход принимаются строки в формате: ВАЛЮТА СУММА

Валюта не чуствительна к регистру. Можно писать rub, RUB, Rub.

Сумма может быть как положительной (500), так и отрицательной (-500)

Для добавления например 500 рублей, необходимо написать: RUB 500

История операций хранится в "Resource\History.txt"

Список поддерживаемых валют хранится в "Resource\Currency.txt"

Текущий баланс хранится в "Resource\CurrencyBalance.txt"

Допущения
---------
Если ввести данные не по шаблону, отобразится уведомление

Если при вводе будет указана неверная валюта, то отобразится уведомление со списком валют.


Команды
-------
При вводе в консоль "q" и нажатии Enter, произойдёт выход из программы

При вводе в консоль "balance" и нажатии Enter, на экране появится текущий баланс по всем валютам, сумма которых != 0.
